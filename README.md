\# Smart Posture Care System



AI-powered IoT posture monitoring system using MPU6050 and XGBoost.



\## Tech Stack

\- Python

\- XGBoost

\- Arduino Uno

\- MPU6050

\- pySerial



\## Features

\- Real-time posture detection

\- 6-axis motion data streaming

\- Calibration \& offset correction

\- Buzzer alert system

\- 91%+ classification accuracy



\## How It Works

MPU6050 → Arduino → Serial → Python → XGBoost → Buzzer Alert
